# Newrosama 개발 로드맵

목표: **3D 아바타 + 다국어(한/일/영) 음성 인식 + AI 대화 + 음성 답변(TTS)**을
갖춘 가상 캐릭터 "뉴로사마"를 GitHub Pages에서 무료로 동작하게 만들기.

진행 방식: 아래 단계를 순서대로 따라가면 매 단계마다 "눈으로 보이는" 결과물이
생기도록 구성했어요. 1~3단계는 이미 이번 커밋에 코드로 구현되어 있어요.

## 🚧 Phase 4 — 진짜 AI 두뇌 연결 (다음 작업)
지금은 `js/brain.js`가 키 없이도 동작하는 샘플 답변을 들려줘요. 진짜 LLM과
대화하려면:
1. Claude API(또는 OpenAI 등) 호출용 **프록시 서버**를 하나 둡니다
   (Vercel/Cloudflare Workers 무료 플랜으로 충분 — 브라우저에 API 키를
   직접 노출하지 않기 위해 꼭 필요해요).
2. 그 프록시가 `{ reply: "..." }` 형태의 JSON을 돌려주도록 만듭니다.
3. 페이지의 ⚙ 설정에서 프록시 URL과 키를 입력 → 저장.
4. `js/brain.js`의 `SYSTEM_PROMPT`를 다듬어 뉴로사마의 성격/말투를 정의합니다.

체크리스트
- [ ] 프록시 서버 코드 작성 + 배포
- [ ] 시스템 프롬프트로 "뉴로사마" 캐릭터 설정 확정 (성격, 호칭, 말투)
- [ ] 대화 맥락(히스토리) 길이/요약 전략 정하기
- [ ] 답변 언어가 사용자의 발화 언어와 일치하는지 확인
